# 🤝 Affiliate Task Protocol

## 🎯 Conversion Focus
- **Simplified UI**: Affiliate landing pages must NOT look like the dashboard. They are marketing pages, not app views.
- **Single Intent**: Every affiliate page has one goal: User Registration.
- **Clear CTA**: The signup button/form must be the most prominent element on the page.

## 🎨 Design Rules
- **High Readability**: Use generous whitespace and large typography.
- **Instructional Flow**: Lead the user from the referral benefit directly to the action.
- **No Clutter**: Remove navigation links or footers that distract from the primary conversion goal.
