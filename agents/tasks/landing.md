# 🚀 Landing Page Task Protocol

## 🔭 Visibility & Layout
- **Maximum Contrast**: Primary text must be high-contrast against background layers.
- **Visual Hierarchy**: Use typography sizes to clearly distinguish between core value props and secondary details.
- **Glassmorphism**: Use subtle glass effects, but ensure they don't break readability on mobile.

## 📈 Conversion Structure
- **Above the Fold**: Value proposition + Clear CTA must be visible without scrolling.
- **Proof Points**: Trading stats and features must be presented with premium icons and clean layouts.
- **Deterministic CTAs**: Buttons must be large, high-intent, and consistent in style.

## 📱 Mobile-First
- Ensure no horizontal scrolling.
- Menu systems must be touch-friendly and high-contrast.
- Font sizes must not drop below 12px for readability.
